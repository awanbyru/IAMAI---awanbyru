IAMAI - awanbyru Project

This is a Next.js application. To view the website, you must run it using the development server.

Opening this HTML file directly in the browser will not work.

Follow these steps:
  1. Open your terminal or command prompt.
  2. Run npm install to install dependencies.
  3. Run npm run dev to start the server.

After starting the server, open your web browser and go to:

http://localhost:3000
